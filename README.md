# Global Climate Records - Data Analysis and Visualization Dashboard

A structured meteorological data processing and visualization project designed to compile, normalize, and display baseline climate telemetry from diverse global monitoring stations. The system structures heterogeneous historical records from the year 2024, tracks critical environmental parameters, and renders an interactive browser interface to filter global weather observations.

## Project Deliverables & Architecture

The historical datasets and user interfaces are organized across separate file formats optimized for analytical evaluations and structural reviews:

- Dynamic Climate Dashboard (.html): A client-side data web application featuring real-time tabular sorting, categorical filtering, and dynamic index visualizations.
- Comprehensive Metric Logs (.csv): A standardized comma-separated dataset containing validated geological coordinates, specific timeframes, and precise environmental variables.

## Dataset Structure & Environmental Parameters

The data processing layer standardizes core atmospheric observations into uniform quantitative metrics across multiple tracking vectors:

- Geospatial Attributes: Structural tagging containing target cities, corresponding countries, and recording timestamps.
- Thermal Matrix: Granular measurement units recording minimum, maximum, and calculated average temperatures in Celsius.
- Atmospheric Telemetry: Continuous monitoring of humidity percentage ratios, active precipitation totals in millimeters, wind direction matrices, and exact wind speed indexes.
- Solar & Cloud Indices: Quantitative tracking of structural UV Index thresholds and cloud cover percentages to classify active regional conditions.

## Technical Implementation & Features

- Zero External Dependencies: Developed entirely using native web scripts and styling elements to guarantee lightweight execution and standalone offline functionality.
- Client-Side Filter Mechanics: Built-in JavaScript filtering logic that enables real-time data screening by localized seasons or matching alphanumeric character strings.
- Interactive Column Sorting: Tabular sorting algorithms mapping structural records dynamically based on chosen numerical fields or geographical parameters.
- Dark-Themed Visual Layout: A modern, low-contrast desktop interface layout built over native CSS variables to ensure high typography legibility and unified data styling.
